I.len()

len()方法返回对象(字符串、列表、元组等)的长度或项目个数

>>>l = [1,2,3,4,5]
>>>len(l)
5

也可以使用__len__()

>>>str1 = '12345'
>>>str1.__len__()
5



II.python中没有 ?: 这样的三目运算符，但也有相应的语法支持

一般支持三目运算符的语言的语法格式如下：
	判断条件（返回布尔值）？为真时的结果:为假时的结果

>>>x = x%2==1?x+1:x	

python中对应的语法格式如下：
	为真时的结果 if 判断条件 else 为假时的结果

>>>x = x+1 if x%2==1 else x
